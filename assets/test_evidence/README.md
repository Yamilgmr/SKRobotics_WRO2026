# Test Evidence

This folder stores real test evidence collected from the physical robot.

## Current Evidence

- [Open Challenge Serial Monitor test - 2026-07-13](open_challenge_serial_monitor_2026-07-13.png)

The Serial Monitor capture shows live Open Challenge state-machine output from the Arduino Mega during a real robot test. It includes straight driving, right-turn behavior, turn adjustment, turn exit, corner counting, and lap counting output.

This evidence is not a replacement for calibrated numeric datasets. Distance, servo, motor, and HuskyLens calibration tables still need to be measured and recorded separately.
