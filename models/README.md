# Models and Fabrication Files

This folder will contain CAD, STL, DXF, or other manufacturing files used to build the vehicle.

Planned files:

- Chassis CAD.
- Steering linkage geometry.
- Sensor mounts.
- Battery holder or battery restraint.
- Motor mount.
- Front ultrasonic bracket.
- Left and right ultrasonic brackets.

For each model, document:

- File name.
- Part purpose.
- Material.
- Printer or fabrication settings.
- Version.
- Reason for design changes.
